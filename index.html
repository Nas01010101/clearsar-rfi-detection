Abstract

We present a detection pipeline for Radio-Frequency Interference (RFI) in Sentinel-1 SAR quicklook imagery, reaching 0.4776 mAP50-95 on the held-out test set of the ESA ClearSAR Track 1 challenge (28th of 172 teams). The pipeline combines a YOLO26x base detector with a pixel-space Normalised Wasserstein Distance (NWD) regression loss, pseudo-label distillation from a multi-source ensemble teacher onto the unlabelled test imagery, multi-resolution test-time augmentation fused by Weighted Box Fusion (WBF), and a selective cross-fold ensemble that drops the domain-overlapping fold.

On a leak-isolated meta-validation set the distilled student source gains +0.077 over a cold baseline — but on genuinely unseen data that source is redundant with the ensemble teacher, and the full stack adds only +0.0021 over submitting the teacher alone. We report both measurements; the gap between them is the main result.

A loss that survives thin boxes

For a 10-px stripe, predicted and target boxes rarely overlap, so CIoU gives no gradient. We blend in a pixel-space NWD: each box becomes a 2-D Gaussian and we measure the Wasserstein-2 distance — smooth even with zero overlap.

// box → Gaussian
μ = (cx, cy)   Σ = diag(w²⁄4, h²⁄4)
// similarity, C = 12.8 px
W₂² = ‖Δμ‖² + ¼(Δw² + Δh²)
NWD = exp(−√W₂² ⁄ C)

On genuinely unseen data the entire distillation–TTA–blend stack adds +0.0021 over the teacher (a ~37× smaller gain) and +0.0056 over a plain 5-fold ensemble. The takeaway: a quality-weighted multi-fold ensemble recovers most of the attainable accuracy on fresh SAR data; the stack is a small, fragile top-up.
